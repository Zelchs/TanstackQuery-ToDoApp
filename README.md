# TanstackQuery-ToDoApp

TanstackQuery-ToDoApp is a simple and efficient to-do list application built with React and Tanstack Query (formerly React Query). This project demonstrates how to manage server-state and local-state seamlessly with Tanstack Query, providing a smooth and responsive user experience.

## Features

- Add, edit, and delete to-do items
- Mark to-do items as completed
- Persist to-do items using a backend API
- Optimistic updates for a responsive UI
- Error handling and retry logic

## Getting Started

### Prerequisites

Before you begin, ensure you have the following installed:

- Node.js (>=14.x)
- npm (>=6.x)

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/Zelchs/TanstackQuery-ToDoApp.git
   cd TanstackQuery-ToDoApp
   ```

2. Install the dependencies:

   ```bash
   npm install
   ```

### Running the Application

To start the development server, run:

```bash
npm start
```
